Introduction 

Qualitia is a Test automation engine that creates, executes, and maintains scriptless automated tests on multiple test automation tools and technologies. This Qualitia Bot (erstwhile Qualitia Offline Package) helps you in executing your test cases without installing Qualitia. You can create and execute same bots on multiple platforms and browsers configuring the bot accordingly.
